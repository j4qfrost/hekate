# Code of Conduct

hekate follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be welcoming, assume good faith, disagree on the substance not the person. Harassment, discrimination, and bad-faith engagement are not tolerated.

## Reporting

To report a violation, email the maintainers at `conduct@<TBD>` (the address will be set when the public reference instance lands at M4). Until then, open a private security advisory on GitHub or DM a maintainer on Bluesky.

Reports are handled in confidence. Outcomes range from a private clarification to a permanent ban from project spaces, depending on severity and pattern.

## Scope

This applies to all hekate project spaces: issue tracker, pull requests, discussions, and any synchronous space (Discord, Matrix, etc.) that uses the hekate name.
